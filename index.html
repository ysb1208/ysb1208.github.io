Project Duration

2022.03~2022.08

Work I Did

  • Implemented an object-grabbing algorithm

Lessons Learned / Improvements

  • Since the rotation angle of the robotic arm varies depending on the object's location, segmentation is more efficient than object detection
  • By improving the grabbing algorithm, the robot can be enhanced to handle objects of various shapes
  • By calculating joint angles using tools such as a linkage simulator and controlling the robotic arm accordingly, the palletizing robot can be used even in narrow spaces
